This project presents a financial model and company valuation for Colgate-Palmolive. The main objective of this project is to understand how financial analysts evaluate a company using historical financial data and forecasting techniques. The model was built in Microsoft Excel and is based on the company’s financial performance over the last five years.

The first step in the project was collecting and organizing the company’s historical financial statements. These include the Income Statement, Balance Sheet, and Cash Flow Statement. The historical data helped identify trends in the company’s revenue growth, operating profit, expenses, assets, liabilities, and cash flows. Understanding these trends is important because they help build reasonable assumptions for future projections.

After organizing the historical data, the next step was financial forecasting. Using the past five years of data, assumptions were made about future growth rates, operating margins, capital expenditure, and working capital changes. Based on these assumptions, the financial statements were projected for the next several years. This step helped estimate how the company might perform in the future.

Once the forecasted financial statements were prepared, the model calculated Free Cash Flow to the Firm (FCFF). FCFF represents the cash flow available to all investors in the company, including both equity investors and debt holders. This cash flow is an important input for company valuation.

The valuation was performed using the Discounted Cash Flow (DCF) method. In this method, the projected future cash flows are discounted to their present value using the Weighted Average Cost of Capital (WACC). The model also calculates a terminal value to estimate the company’s value beyond the forecast period. By combining the present value of forecasted cash flows and the terminal value, the model estimates the intrinsic value of the company.

The purpose of this project is to practice financial modeling, improve Excel skills, and develop a deeper understanding of company valuation methods used in finance and investment analysis.
